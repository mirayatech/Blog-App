### 💬  &nbsp; Blog-App 
A blog app built with react, typescript and firebase 


### 🧠 - What I learned 

* Create a Collection in Firestore
* Delete and update a document from a collection 
* Integrate Google Authentication with Firebase
* And much more

### 🛠️ - Dependencies 

* `React`
* `Firebase`
* `TypeScript`
* `Tailwind CSS`
* `Build tool: Vite`
